# Advance 聚合SDK（鸿蒙版本）接入文档

sdk版本：v1.0

## 1.SDK集成与初始化

### 引入SDK

参考demo工程，将har文件放入工程libs目录中，刷新项目即可使用聚合广告api方法

注：如果之前接入了har或者本次为替换har包更新版本，则需要先删除模块的 oh_modules目录，避免更新har包失败。



### 权限申请

广告依赖部分设备信息进行转化，需要媒体在module.json5文件中添加以下权限：

```

// 必选权限
{
  "name": "ohos.permission.INTERNET" //访问网络  必选
},

// 可选权限
{
  "name": "ohos.permission.GET_NETWORK_INFO"  //访问网络信息 可选，影响转化
},
{
  "name": "ohos.permission.GET_WIFI_INFO" //访问网络信息 可选，影响转化
},
{
  "name": "ohos.permission.APPROXIMATELY_LOCATION" //模糊地理位置，5公里 可选，影响转化
},
{
  "name": "ohos.permission.LOCATION" //获取米级别位置 可选，影响转化
},
{
  "name": "ohos.permission.APP_TRACKING_CONSENT" //获取oaid  可选，影响转化
}

```

### 初始化SDK

```
    AdvanceSDK.initSDK(this.context, 'advance应用id')
```

### 初始化相关API参考（AdvanceSDK）

| 方法名| 	方法介绍
|:------------- |:---------------|  
| initSDK(context: Context, appId: string) | 初始化聚合SDK， 必须
| debug() | 调试模式，调用后即代表打开调试模式。默认为关闭状态
| setPrivacyController(advPri: AdvancePrivacyController)  | 设置隐私合规
