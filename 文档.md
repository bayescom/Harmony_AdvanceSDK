# Advance 聚合SDK（鸿蒙版本）接入文档

版本号：v1.0

## 1.SDK集成与初始化

### 引入SDK

参考demo工程，将har文件放入工程libs目录中，在项目`oh-package.json5`文件中引入har包，刷新项目即可使用聚合广告api方法

```
  "dependencies": {
    //引入聚合SDK，其中包含了穿山甲的har包
//    穿山甲har中方法无法被app直接调用，如果想要在项目中使用穿山甲SDK方法，且不想增大包体积，可以利用ohpm的override机制，在单独引入穿山甲独立的har包，并在根目录oh-package.json5 文件中使用overrides声明使用的穿山甲版本。注意此举存在版本不兼容风险，谨慎选择
    "advSDK": "file:libs/advance_v1.0.har"
  }
```

注：如果之前接入了har或者本次为替换har包更新版本，则需要先删除模块的 oh_modules目录，避免更新har包失败。

| 穿山甲   | v6.4.0.1 | 支持  | 支持     | 支持    |

### 权限申请

广告依赖部分设备信息进行转化，需要媒体在module.json5文件中添加以下权限：

```

// 必选权限
{
  "name": "ohos.permission.INTERNET" //访问网络  必选
},

// 可选权限
{
  "name": "ohos.permission.GET_NETWORK_INFO"  //访问网络信息 可选，影响转化
},
{
  "name": "ohos.permission.GET_WIFI_INFO" //访问网络信息 可选，影响转化
},
{
  "name": "ohos.permission.APPROXIMATELY_LOCATION" //模糊地理位置，5公里 可选，影响转化
},
{
  "name": "ohos.permission.LOCATION" //获取米级别位置 可选，影响转化
},
{
  "name": "ohos.permission.APP_TRACKING_CONSENT" //获取oaid  可选，影响转化
}

```

- AdvancePrivacyController 隐私合规类字段详细说明

```
abstract class AdvancePrivacyController {
  /**
   * app在项目中配置权限。在适当时机申请权限，然后用户授权。
   * 1：如果isCanUseAppTrackingConsent()返回false，当sdk需要oaid时，则使用getDevOaid()返回的值。
   * 2：如果isCanUseAppTrackingConsent()返回true。当sdk需要oaid时，先去检查权限是否已经得到用户授权，如果用户已经授权则获取系统oaid。如果用户未授权，则使用getDevOaid()返回的值。
   * @returns 是否允许SDK获取系统oaid
   */
  isCanUseAppTrackingConsent(): boolean {
    return true
  }

  /**
   *
   * @returns 媒体传入的oaid
   */
  getCusOaid(): string | undefined {
    return undefined
  }


  /**
   * app在项目中配置权限。并在合适时机动态申请权限，用户授权。
   * 1：当isCanUseLocation()返回false。当sdk需要地理位置信息时，使用  getCusLocation()返回的值。
   * 2：当isCanUseLocation()返回true。当sdk需要地理位置信息时，先去检查是否已经得到用户授权，如果是则获取系统地理位置信息，如果否则使用  getCusLocation()返回的值。
   * @returns 是否允许SDK获取系统地理位置信息
   */
  isCanUseLocation(): boolean {
    return true
  }


  /**
   *
   * @returns 媒体传入的地理位置信息
   */
  getCusLocation(): BYLocation | undefined {
    return undefined
  }

  /**
   * 是否允许SDK使用ohos.permission.GET_WIFI_INFO权限对应的信息。
   * 返回true，如果应用申请了ohos.permission.GET_WIFI_INFO权限，sdk则可以使用
   * 返回false，sdk则不可以使用。
   * @returns
   */
  isCanUseWifiState() {
    return true
  }

  /**
   * 媒体isCanUseWifiState()返回false时，SDK使用getMacAddress()的返回值作为mac地址
   * @returns
   */
  getMacAddress(): string | undefined {
    return undefined
  }
}
```

## 2.广告加载

### 2.1 开屏广告

发起请求

```
//设置请求参数
let opt = new AdvanceOption(this.adID)
//初始化开屏
this.splash = new AdvanceSplash(opt)
//监听广告回调事件
this.splash.setListener({
  onShow: (): void => {
    logToast(`onShow`)
  },
  onClose: (closeType: AdvanceCloseType): void => {
    logToast(`onClose , closeType = ${closeType}`)
  },
  onLoaded: (supplier: AdvanceSupplier): void => {
    logToast(`onLoaded , supplier : ${JSON.stringify(supplier)}`)
  },
  onClick: (): void => {
    logToast(`onClick`)
  },
  onRenderSuccess: (): void => {
    logToast(`onRenderSuccess`)
  },
  onRenderFail: (err: AdvanceError): void => {
    logToast(`onRenderFail , ${err.toString()}`)
  },
  onError: (err: AdvanceError): void => {
    logToast(`onError , err = ${err.toString()}`)
  }
})
//加载广告策略
this.splash.loadStrategy()
```

  /**
   * Feed自渲染广告
   * @param ad
   */
  @Builder
  private buildAdLayout(ad: AdvanceNativeData, clickViewIds: ArrayList<string> = new ArrayList(),
    creativeViewIds: ArrayList<string> = new ArrayList(), rootComponentId: string = util.generateRandomUUID()) {
    Column() {
      Row() {
        Row() {
          //广告主Icon
          Image(ad.getIconUrl())
            .height(32)
            .width(32)
            .borderRadius(5)
            .margin({ right: 5 })
            .id(this.generateViewId(clickViewIds)) //设置组件Id，需要全局保证唯一性，涉及计费

          //描述内容
          Text(ad.getDescription())
            .fontSize(12)
            .textAlign(TextAlign.Start)
            .maxLines(2)
            .textOverflow({ overflow: TextOverflow.Ellipsis })
            .id(this.generateViewId(clickViewIds)) //设置组件Id，需要全局保证唯一性，涉及计费
        }
        .height(35)
        .width("85%")
        .justifyContent(FlexAlign.Start);

        Row() {
          //dislike 按钮
          Image($r("app.media.icon_dislike")).onClick(() => {

            ad.showDislikeDialog(this)
          })
            .height(16)
            .width(16)
            .borderRadius(8)
        }.height(35)
        .width("10%")
        .justifyContent(FlexAlign.End);
      }
      .justifyContent(FlexAlign.SpaceBetween)
      .height(35)
      .width("100%")
      .padding({ left: 10, right: 10, bottom: 4 })

      //视频|图片区域
      Stack({ alignContent: Alignment.BottomStart }) {
        if (ad.isVideo()) {
          NodeContainer(ad.getAdComponent())
        } else if (ad.getImageUrlList().length > 0) {
          this.buildImageLayout(ad, creativeViewIds)
        }

        Flex({ direction: FlexDirection.Column, justifyContent: FlexAlign.End }) { // 子组件在容器主轴上行布局
          if (typeof ad.getAdLogo() === "string") {
            Text(ad.getAdLogo())
              .fontSize(6)
              .height(18)
              .width(30)
              .fontColor(Color.White)
              .backgroundColor("#40ffffff")
              .align(Alignment.Center)
              .padding({ left: 2 })
          } else {
            Image(ad.getAdLogo()).height(20).width(30).objectFit(ImageFit.Auto).onClick((event: ClickEvent) => {
              ad.onAdClick(2, event);
            })
          }
        }.width("100%")
        .height(20)

      }.height(200)

      Row() {
        //标题
        Text(ad.getTitle())
          .fontSize(12)
          .textAlign(TextAlign.Center)
          .id(this.generateViewId(creativeViewIds)) //设置组件Id，需要全局保证唯一性，涉及计费


        //创意按钮
        Button(ad.getButtonText(), { type: ButtonType.Normal })
          .fontSize(12)
          .borderRadius(5)
          .id(this.generateViewId(creativeViewIds)) //设置组件Id，需要全局保证唯一性，涉及计费

      }
      .justifyContent(FlexAlign.SpaceBetween)
      .height(35)
      .width("100%")
      .padding({
        top: 5,
        bottom: 5,
        left: 10,
        right: 10
      })
    }
    .backgroundColor("#08000000")
    .id(rootComponentId) //设置广告容器组件rootId
    .onAppear(() => {
      let advInter = new AdvanceNativeInteractionModel()
      advInter.rootAdComponentId = rootComponentId
      advInter.uiContext = this.getUIContext()
      advInter.clickViewIds = clickViewIds
      advInter.creativeViewIds = creativeViewIds
      //注册计费事件
      ad.registerViewForInteraction(advInter)
    })
  }

  /**
   * 图片类型广告布局
   * @param ad

  @Builder
  buildImageLayout(ad: AdvanceNativeData, creativeViewIds: ArrayList<string>) {
    if (ad.getImageUrlList().length > 1) {
      Row({ space: 10 }) {
        ForEach((ad.getImageUrlList() ?? new ArrayList<string>()).convertToArray(), (url: string) => {
          Image(url)
            .objectFit(ImageFit.Cover)
            .width(((100 / (ad.getImageUrlList().length ?? 0) - 1) + "%"))

            .id(this.generateViewId(creativeViewIds)) //设置组件Id，需要全局保证唯一性，涉及计费;
        })
      }.justifyContent(FlexAlign.SpaceBetween).width("100%").alignItems(VerticalAlign.Center).height("100%")
    } else if (ad.getImageUrlList().length == 1) {
      Image(ad.getImageUrlList()[0])
        .objectFit(ImageFit.Cover)
        .id(this.generateViewId(creativeViewIds)) //设置组件Id，需要全局保证唯一性，涉及计费
    }
  }

  generateViewId(savedViewIds: ArrayList<string>): string {
    let viewId = util.generateRandomUUID()
